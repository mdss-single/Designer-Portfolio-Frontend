Mitchell Hayes is a multipurpose HTML5 template that contains 14 pages. The layout is ideal for a artist portfolio, male and female designers, design studios and much more. The template comes with beautiful vector icons that makes all the pages to look amazing across the retina devices.

Gulp and customize

For customize your template you need use gulp task runner.

If you don't know what is it and how to use it, read that short article: https://css-tricks.com/gulp-for-beginners/


How to install gulp dependency packages


Open a command line in the project folder (a folder that includes gulpfile.js file) and run gulp command. It's all.

Main commands for gulp:

Command Name Description
gulp

Main command

- all *.js files from src/js/libs folder will be merged and compressed to one file to dist/js/plugins.js file with sourcemaps

- all *.js files from src/js folder just be copied to dist/js folder

- all *.css files from src/style/libs folder will be merged and compressed to one file to dist/css/plugins.css file with sourcemaps

- all *.styl files from src/style folder will be compiled, merged and compressed to one file to dist/main.min.css file with sourcemaps and to dist/main.css with no compress

- all files from src/i folder will be copied to dist/i folder

- all files from src/fonts folder will be copied to dist/fonts folder

- starting local webserver for you compiled project

- starting watch command for watching file changes. For example, if you change src/style/main.styl file gulp will see that and automatically compile styles and reload your browser localhost tab. Gulp watching for changes in this folders:

  • src/i/ - for all files
  • src/fonts/ - for all files
  • src/js/ - for all *.js files
  • src/js/libs/ - for all *.js files
  • src/style/ - for all *.styl files
  • src/style/css - for all *.css files
  • src/ - for all html files
gulp-build

It is like a gulp command, but without starting webserver and watching for changes
gulp-final

It is like gulp-build command, but without sourcemaps files for compiled css and js

How to add some plugin or library


You can simply add any library or plugin. For *.js files copy them to src/js/libs folder. For *.css files copy them to src/style/libs folder.

And they will be merged and compressed to js/plugins.js file and css/plugins.css file respectively.
